# Nexo Pro connector for PHP

## Install using composer ##

    composer require stilmark/nexo-pro-php

## Documetation ##

https://stilmark-projects.gitbook.io/nexo-pro-for-php/

### Supported methods ###

**Portfolio**
- getAccountSummary
- getPairs
- getQuote

**Orders**
- placeOrder
- placeAdvancedOrder
- placeTriggerOrder
- getOrder
- cancelOrder
- cancelAllOrders

**History**
- getOrderHistory
- getTradeHistory

**Transaction**
- getTransaction
